**Para:** Desarrollador que quiere construir su Knowledge Base con IA  
**Tiempo:** 5 minutos de lectura + configuración guiada con Claude Code

---

## 🎯 ¿QUÉ VAS A CONSTRUIR?

Una aplicación personal tipo **Recall/NotebookLM** que:
- 📥 Captura contenido de YouTube, TikTok, Twitter, web
- 🤖 Resume y categoriza automáticamente con IA
- 🔍 Búsqueda semántica inteligente
- 💬 Chat con tu contenido (RAG)

---

## ⚡ 3 PASOS PARA EMPEZAR

### PASO 1: Abrir en VS Code (1 minuto)

1. Descarga la carpeta `PROYECTO_KNOWLEDGE_BASE_AI`
2. Abre VS Code
3. `File` → `Open Folder` → Selecciona la carpeta

### PASO 2: Abrir Claude Code (1 minuto)

1. Presiona `Cmd+Shift+P` (Mac) o `Ctrl+Shift+P` (Windows)
2. Escribe: `Claude Code: Open`
3. Presiona Enter

### PASO 3: Cargar Contexto (3 minutos)

1. Abre el archivo `PROMPT_INICIAL_CLAUDE_CODE.md`
2. Selecciona TODO (`Cmd+A` / `Ctrl+A`)
3. Copia (`Cmd+C` / `Ctrl+C`)
4. Pega en Claude Code (`Cmd+V` / `Ctrl+V`)
5. Presiona Enter
